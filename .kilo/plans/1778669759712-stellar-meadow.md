### 1. Update setup.sh - Auto-detect SSH port

**Current state:** setup.sh requires `-s` flag for SSH port; if omitted, no SSH firewall rule is added.

**Changes:**
- Before parsing `-s` argument, read `/etc/ssh/sshd_config` to detect actual SSH port
- If port is found and no `-s` provided, use detected port as default
- Store detected port in variable for UFW rule

**Implementation approach:**
```bash
DETECTED_SSH_PORT=$(grep -E "^Port" /etc/ssh/sshd_config 2>/dev/null | awk '{print $2}' | head -1)
DETECTED_SSH_PORT="${DETECTED_SSH_PORT:-22}"
```
